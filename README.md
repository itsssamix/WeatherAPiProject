Uma aplicação simples que desenvolvi em Python com Flask para consultar a previsão do tempo de qualquer cidade consumindo a API do OPENWEATHERMAP.

O projeto demonstra a criação de rotas, manipulação de formulários e integração com serviços externos via requisições HTTP (requests).

Tecnologias utilizadas:
-Python 3
-Flask
-Biblioteca Requests
-API Externa: OpenWeatherMap

Requisitos e execução:
-Será necessário uma CHAVE DE API do OpenWeatherMap. É possível conseguir através da criação de uma conta gratuita.

Instalação após clonagem do repositório:
-Pip install Flask requests

Localize a linha no arquivo Python (main.py) onde a chave da API está definida e substitua a string genérica pela sua chave real.
-chave_api = 'SUA_CHAVE_AQUI

Execução do servidor: 

-python main.py
